# Lisp Interpreter 

An interpreter for a lisp like language written in Haskell.


## Features:
- [x] Function definitions
- [x] Print Output 
- [x] first class functions  
- [x] conditions 
- [x] arithmetic
- [x] int and bool types
- [x] comments
- [x] lists
- [ ] float, char and string types 
- [ ] More IO
